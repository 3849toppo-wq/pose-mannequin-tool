# pose-mannequin-tool

**スマホ用 動画AIポーズ参考ツール（最小テスト版）**

## 使い方

1. このリポジトリをGitHubで開く
2. index.htmlをタップして「Raw」またはブラウザで開く

## GitHub Pagesで公開してスマホでアプリのように使う方法（スマホから設定）

このリポジトリは現在**private**です。

GitHub Pages（無料プラン）で公開URLを作るには、**リポジトリをpublicにする**必要があります。

### 手順（スマホブラウザ or GitHubアプリで）

1. このリポジトリページを開く
2. 上部の「Settings」（設定）をタップ
3. 「General」または「Danger Zone」セクションまでスクロール
4. 「Change visibility」または「Make public」をタップ
5. 「I understand...」にチェック → 「Make public」を実行

※ ツールに個人情報は一切入っていないので、public化しても問題ありません。

### GitHub Pagesを有効化

1. Settingsページに戻る
2. 左メニューから「Pages」をタップ
3. 「Source」のドロップダウンで **Deploy from a branch** を選択
4. Branch: **main** / フォルダ: **/ (root)** を選択
5. 「Save」をタップ

### 公開URL

設定完了後、数分で以下のURLでアクセス可能になります：

**https://3849toppo-wq.github.io/pose-mannequin-tool/**

このURLをスマホのホーム画面に追加すれば、アプリのように使えます。
